# Windows API Fundamentals

## Variable Types
- DWORD - 32 bit unsigned integer
- HANDLE - States an object managed by the OS
- HMODULE - Handle to a module (DLL or EXE)
- LPCSTR/PCSTR - Pointer to constant strings (const char*)
- LPSTR/PSTR - Pointer to readable string (char*)
- LPCWSTR/PCWSTR - Pointer to constant Unicode string
- LPWSTR/PWSTR - Pointer to Unicode string (wchar*)
- PVOID - Void pointer to any data type
- ULONG_PTR - Unsigned integer sized to OS ARCH (32/64 bit)

## ERROR CODES
- 5 - ERROR_ACCESS_DENIED
- 2 - ERROR_FILE_NOT_FOUND
- 87 - ERROR_INVALID_PARAMETER
