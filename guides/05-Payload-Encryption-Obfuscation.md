# Payload Encryption & Obfuscation

## Obfuscation Techniques
- Dead Code - Add non-functional code to confuse analysis
- Code Reordering - Rearrange instruction order
- API Hashing - Hide strings and API calls via hashing
- String Encryption - Encrypt constant strings in binary
- Control Flow Flattening - Flatten loops and conditions

## Encryption Methods
- XOR - Simple byte-wise encryption
- AES - Strong encryption for payloads
- RC4 - Stream cipher (faster but weaker than AES)
- Custom Algorithms - Unique encryption to evade detection
